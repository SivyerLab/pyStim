pyStim.main(stim_list, verbose=True)[source]¶

Function to create stims and run program. Creates instances of stim types, and makes necessary calls to animate stims and flip window.

Parameters

  • stim_list (list) – list of StimInfo classes.

  • verbose (boolean) – whether or not to print stim info to console.

Return fps, count_elapsed_time, time_stamp

return stats about last run. If error was raised, fps is the error string, and count_elapsed_time is ‘error’.

Function to write information about stims to file.

Parameters

  • count_reps – Elapsed reps.

  • reps – Total possible reps.

  • count_frames – Elapsed frames.

  • num_frames – Total possible frames.

  • elapsed_time – Elapsed time

  • stim_list – List of stims that ran.

  • to_animate – List of stims animated (includes annuli)

  • time_at_run – Time at which stims were run
