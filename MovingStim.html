Moving Stim¶

class pyStim.MovingStim(**kwargs)[source]¶

Bases: pyStim.StaticStim

Class for stims moving radially inwards. Overrides several methods.

draw_times()[source]¶

Determines during which frames stim should be drawn, based on desired delay and duration times. Overrides super method.

Returns

last frame number as int

animate(frame)[source]¶

Method for animating moving stims. Moves stims appropriately, then makes call to animate of super.

Parameters

frame (int) – current frame number

gen_pos()[source]¶

Makes calls to gen_start_pos() and gen_pos_array() with proper variables to get new array of position coordinates.

gen_start_pos(direction)[source]¶

Calculates starting position in x, y coordinates on the starting radius based on travel direction.

Parameters

direction – starting position on border of frame based on travel

Returns

starting position on border of frame based on travel angle origin

gen_pos_array(start_x, start_y, num_frames, angle)[source]¶

Creates 2 arrays for x, y coordinates of stims for each frame.

Adapted from code By David L. Morton, used under MIT License. Source: https://code.google.com/p/computational-neuroscience/source/browse/trunk/projects/electrophysiology/stimuli/randomly_moving_checkerboard_search.py/#40

Parameters

  • start_x – starting x coordinate

  • start_y – starting y coordinate

  • num_frames – number of frames stim will travel for

  • angle – travel direction

Returns

the x, y coordinates of the stim for every frame as 2 arrays
