Static Stim¶

class pyStim.StaticStim(**kwargs)[source]¶

Bases: pyStim.StimDefaults

Class for generic non moving stims. Super class for other stim types. Stim object instantiated in make_stim, and drawn with calls to animate().

make_stim()[source]¶

Creates instance of psychopy stim object.

draw_times()[source]¶

Determines during which frames stim should be drawn, based on desired delay and duration times.

Returns

last frame number as int

animate(frame)[source]¶

Method for drawing stim objects to back buffer. Checks if object should be drawn. Back buffer is brought to front with calls to flip() on the window.

Parameters

frame (int) – current frame number

gen_rgb()[source]¶

Depending on color mode, calculates necessary values. Texture color is either relative to background by specifying intensity in a certain channel, or passed as RGB values by the user.

Returns

tuple of high, low, delta, and background

gen_size()[source]¶

Calculates sizes of various sims.

Returns

size of stim, as float for circles/annuli and height width tuple for other shapes

gen_mask()[source]¶

Determines the mask of the stim object. The mask determines the shape of the stim. See psychopy documentation for more details.

Returns

mask of the stim object, as a string

gen_texture()[source]¶

Generates texture for stim object. Textures are 3D numpy arrays (size*size*4). The 3rd dimension is RGB and Alpha (transparency) values.

Returns

texture as numpy array
